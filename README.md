Tree Menu editor
============================

Tree Menu editor use Yii 2 Basic Project Template

INSTALLATION
------------

Clone to your PC, extract and run

```
    composer update
```

CONFIGURATION
-------------

Database.
Edit the file `config/db.php` with real data, now use:

```php
    'dsn' => 'mysql:host=localhost;dbname=dbtreemenu',
```
run migration

```
    php yii migrate
```
